# **[ Pandas ][ 데이터분석 ] 온라인 쇼핑몰 데이터 분석 및 가공**

---

가상 온라인 쇼핑몰 데이터(csv)를 통해, 데이터 분식 및 가공

### 컬럼
* InvoiceNo: 주문 번호
* StockCode: 상품 코드
* Description: 상품 설명
* Quantity: 주문 수량
* InvoiceDate: 주문 날짜
* UnitPrice: 상품 가격
* CustomerID: 고객 아이디
* Country: 고객 거주지역(국가)

-----------


1. 데이터 전처리 진행
2. 고객별 총 지출비용 데이터 컬럼 생성
3. 전체 매출 ( 요일, 월, 시간대별 )
4. 국가별 매출 및 그래프 시각화
5. 우수고객, 판매상품 top 10
